## Project Overview
This repository contains the shell scripts for "The Open Source Audit" capstone project. The chosen open-source software for this audit is Python.

## Prerequisites & Dependencies
* A Linux environment (e.g., Ubuntu, Fedora, or WSL on Windows)
* Bash shell

## Shell Scripts
*(Detailed execution instructions will be added here as the scripts are developed.)*

### 1. System Identity Report
* **Description:** Introduces the Linux system, displaying OS, kernel, user, uptime, and licensing information.
* **How to run:** `bash script1.sh`

### 2. FOSS Package Inspector
* **Description:** Checks if Python is installed on the system, retrieves its version, and outputs a brief description.
* **How to run:** `bash script2.sh`

### 3. Disk and Permission Auditor
* **Description:** Loops through key system directories to report disk space usage, ownership, and permissions.
* **How to run:** `bash script3.sh`

### 4. Log File Analyzer
* **Description:** Scans a log file for specific keywords (like ERROR or WARNING) and prints a summary.
* **How to run:** `bash script4.sh /path/to/logfile`

### 5. Open Source Manifesto Generator
* **Description:** An interactive script that asks three questions to generate a personalized open-source philosophy text file.
* **How to run:** `bash script5.sh`
